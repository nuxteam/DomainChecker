Разработка админ панели для автоматического мониторинга доступности доменов

Окружение и технологии
Backend: Laravel 12+
База данных: MySQL 8+ / Postgres 14+
Frontend: Blade(Laravel Views) или React / Vue (будет плюсом)
Docker: (будет плюсом)
Система контроля версий: Git (GitHub/GitLab)
